Hardware Fingerprint

Use this feature to specifically change your machine's:
1. Standard HW Fingerprint
2. Enhanced HW Fingerprint

Enter a valid value in the appropriate text box, or use both if necessary,
in the format: XXXX-XXXX

where 'X' above is a valid hexadecimal digit (i.e. 0 - 9, A - F).

You would normally do this (BEFORE executing a target) when presented with a name / key pair dialog window which is tied to a specific hardware id. In this case, you can override this hardware id at run time. Armageddon will dynamically change this information to what you provided in the target process's memory.

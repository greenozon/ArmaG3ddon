Refresh

Press this button to Re-execute the current application, start a new session (Refresh) without having to re-launch (restart) Armageddon each and every time, or simply to reset options.

The current application (if exists) will be terminated, and the log window cleared. All options and internal variables will be initialized to their default values.

Note: If you want to save the log to a file, press "Save" logfile prior to pressing (selecting) this option!!

If reanalyzing a target dump via Nanofixer, it will be necessary to restart Armageddon!

For newer versions of Armadillo, it may be necessary to restart this tool!!
